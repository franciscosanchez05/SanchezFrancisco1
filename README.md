# Proyecto de Evaluacion GitHub y Markdown
## Descripcion
Este repositorio forma parte de una **actividad de evaluacion** cuyo objetivo es comprobar el uso de:
- Git y GitHub
- Markdown
- Buenas prácticas de documentacion técnica
El documento sirve como ejemplo de un `README.md` **bien estructurado**, claro y profesional.
